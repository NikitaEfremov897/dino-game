# dino-game
Dino game

Простая игра в стиле Google Dino, реализованная на Pygame.

## Особенности
- Прыжок по пробелу (включая двойной прыжок)
- Автоматический режим (клавиша Q)
- Пауза (P)
- Фон меняется каждые 30 очков (светлый ↔ тёмный)
- Сохранение рекорда
- Ускорение со временем

## Как запустить
1. Установите зависимости: `pip install pygame`
2. Положите `dino.png` и `cactus.png` в папку с игрой
3. Запустите: `python main.py`
